## Lab Topics

| Lab | Title | Key Techniques |

| 1 | Text Cleaning & Tokenization | Tokenization, Stemming, Lemmatization, Stop-word Removal |

| 5 | Text Classification with Deep Learning *(optional)* | Neural Networks, Embedding Layers |
